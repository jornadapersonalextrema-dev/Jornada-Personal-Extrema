# Jornada Personal Extrema

MVP de pesquisa, diagnóstico, leads e mensagens persuasivas para WhatsApp.

## Rodar localmente

```powershell
npm install
Copy-Item .env.example .env.local
npm run dev
```

## URLs

- `/pesquisa`
- `/admin/pesquisa-diego?token=SEU_TOKEN`
- `/admin/pesquisa-diego/leads?token=SEU_TOKEN`

## Supabase

Execute `supabase/schema.sql` no SQL Editor do Supabase e configure as variáveis de ambiente.
